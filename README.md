# [癒しのデスクトップ常駐型マスコット「カニ炒飯（大盛り）」]

## 📌 概要
大学のJavaの講義の最終課題として作成。
講義で学んだ内容に加えてGUIを用いて、という条件だったので、視覚的に楽しいものを作ってみました。

## 🛠 使用言語・ツール
- 言語: Java
- 開発環境: VS Code

## 🚀 使い方
TsukamotoShota1J22F100.java を実行すると画面上にカニ炒飯が表示されます。

実行するには、
javac TsukamotoShota1J22F100.java
java TsukamotoShota1J22F100
と入力します。

ドラッグ&ドロップで画面上の好きな位置に移動することができます。
満足したらカニ炒飯をダブルクリックするとプログラムが終了します。

## ✨ こだわり・苦労した点
– OS標準の枠を排除: 「デスクトップに存在している感」を出すため、敢えてウィンドウのタイトルバーを非表示にし、背景を完全透過させる実装を行いました。
– 独自の操作体系: マウスドラッグによる移動や、ダブルクリックでの終了など、マスコットとして自然な操作感をJavaのイベントリスナーで実現しました。
– AIを活用したアセット制作: 著作権をクリアしつつ好みのビジュアルを得るため、生成AIを活用してドット絵を作成しました。
